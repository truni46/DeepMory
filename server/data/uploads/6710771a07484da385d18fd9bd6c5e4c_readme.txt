Để chạy thực nghiệm mô hình PTPCG, GIT, Doc2EDAG, vui lòng thực hiện các bước sau:

cd PTPCG_GIT_Doc2EDAG
cd Data
python stat.py
nohup bash scripts/run_{model_name}_vlex_withtgg.sh 1>Logs/{model_name}_VLEX_with_trigger.log 2>&1 & tail -f Logs/{model_name}_VLEX_with_trigger.log

Ví dụ: muốn chạy mô hình Doc2EDAG, ta chạy lệnh:
nohup bash scripts/run_doc2edag_vlex_withtgg.sh 1>Logs/Doc2EDAG_VLEX_with_trigger.log 2>&1 & tail -f Logs/Doc2EDAG_VLEX_with_trigger.log

Để chạy thực nghiệm mô hình DCFEE-O, DCFEE-M, vui lòng thực hiện các bước sau:

cd DCFEE
cd code/sentence_level_EE
python main.py --dataset vlex -mode {mode}

Ví dụ: muốn chạy mô hình DCFEE-O, ta chạy lệnh:
python main.py --dataset vlex -mode O
